Вытаскиваем из изображений тканей самые простые признаки (баланс яркости, резкость изменения яркости на волокнах, энтропию), на основе этих признаков строим 100 деревьев и используем случайный лес, чтобы улучшить качество их классификации. Простой ML подоход.

LB1 - визуализация распределения
LB2 - дробление на валидационную (3 фолда) и обучающую выборки
LB3 - построение матрицы ошибок
